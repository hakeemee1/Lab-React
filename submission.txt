1-  Button to start the game => เป็นปุ่มที่เมื่อกดเเล้วสามารถที่จะเปลี่ยนข้อความในตัวมันได้ โดยใช้ความสามารถของ React Hook ที่ชื่อว่า useState 
    โดยจะมีการ setState ใน div ผ่าน onClick เมื่อกดเเล้วจะเรียกใช้ setReady เพื่อเปลี่ยนข้อความเป็น "Game started" เเละยังมีการ consol.log 
    เพื่อให้เเสดงข้อความ "Game started and please enjoy ^^"

2-  สร้าง component Timer ขึ้นมาเพื่อให้มีการจับเวลาในการเริ่มต้นเกมส์เเละผู้เล่นต้องมีการกด "หยุด" เพื่อดูเวลาที่ตัวเองใช้ในการเล่นเกมส์ 
    โดยใช้ความสามารถของ React Hook 2 ตัวด้วยกัน นั่นคือ useState เเละ UseEffect เเละใช้ function setinterval เเละ clearinterval 
    ในการสั่งทำให้เวลาเดินไปต่อเนื่อง โดยจะมีเงื่อนไขหากผู้เล่นกดไปที่ "หยุด" เวลาจะหยุด เเละเมื่อผู้เล่นกดคำว่า "เริ่มใหม่" ก็จะเรียกใช้ function restart 
    นั่นคือ setState ของ Minutes เเละ seconds เป็น 0 นั่นเอง
